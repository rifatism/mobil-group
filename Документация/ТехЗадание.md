**ТЗ №1. Корпоративный сайт компании**

**Цель проекта**
Разработать современный корпоративный сайт, который:
- отражает корпоративную культуру компании;
- презентует компанию и ее ценности;
- демонстрирует продукты и услуги;
- служит инструментом привлечения клиентов, партнеров и кандидатов.

**Основные задачи**

1. Разработка структуры сайта
Предусмотреть следующие разделы: Главная страница, О компании, Корпоративная культура, Продукты и услуги, Кейсы и проекты, Новости и статьи, Карьера, Контакты, Форма обратной связи.

2. Дизайн
- Современный адаптивный дизайн.
- Соответствие фирменному стилю компании.
- Корректное отображение на ПК, планшетах и мобильных устройствах.
- Использование фото, видео и графических материалов для демонстрации корпоративной культуры.

3. Функционал
- CMS для самостоятельного управления контентом.
- Управление новостями и публикациями.
- Управление вакансиями.
- Формы обратной связи.
- Интеграция с CRM (при необходимости).
- SEO-оптимизация.

4. Результат
- Полностью функционирующий корпоративный сайт.
- Документация по администрированию.
- Передача исходных кодов и доступов.

**ТЗ №2. Система управления корпоративной документацией и базой знаний**

**Цель проекта**
Создать единую информационную среду для хранения, актуализации и поиска корпоративной документации. Допускается миграция существующей базы знаний на новую платформу.

**Основные задачи**

1. Анализ текущей системы
- Аудит действующей базы знаний.
- Анализ структуры данных.
- Оценка возможности миграции существующего контента.
- Подготовка плана переноса.

2. Внедрение новой платформы
Система должна обеспечивать:
- хранение документов;
- создание и редактирование статей;
- управление версиями документов;
- разграничение прав доступа;
- поиск по содержимому;
- согласование и публикацию документов.

3. Миграция данных
- Перенос существующей базы знаний.
- Сохранение структуры разделов.
- Сохранение вложений и файлов.
- Проверка корректности переноса.

4. Наполнение системы
Создание разделов: Регламенты, Инструкции, Политики компании, Шаблоны документов, База знаний сотрудников, FAQ.

5. Интеграции (при необходимости)
- Active Directory / LDAP
- Корпоративная почта
- Корпоративный портал
- Системы управления задачами

6. Результат
- Введенная в эксплуатацию база знаний.
- Перенесенный исторический контент.
- Настроенные роли пользователей.
- Инструкция для администраторов и пользователей.

**ТЗ №3. Информационная система для клиентов с интеграцией мониторинга транспорта**

**Цель проекта**
Разработать клиентский портал, интегрированный с системой мониторинга транспорта, предоставляющий клиентам доступ к информации о перевозках в режиме реального времени.

**Основные задачи**

1. Создание личного кабинета клиента
Функционал: авторизация пользователей, управление профилем, разделение доступа по ролям, история операций.

2. Интеграция с системой мониторинга транспорта
Система должна получать данные: местоположение транспортных средств, статус перевозки, маршрут, время прибытия, отклонения от маршрута, уведомления о событиях.

3. Разработка дашбордов
Для клиента должны отображаться: текущее количество перевозок, транспорт в пути, выполненные перевозки, проблемные рейсы, статистика по периодам, карта с отображением транспорта.

4. Отчетность
Формирование отчетов: по перевозкам, по транспорту, по маршрутам, по соблюдению сроков доставки.
Выгрузка: PDF, Excel, CSV.

5. Уведомления
- Email-уведомления.
- Push-уведомления.
- Оповещения о задержках и изменениях статуса.

6. Требования к безопасности
- Авторизация и аутентификация пользователей.
- Журналирование действий.
- Разграничение прав доступа.
- Защита персональных данных.
- Шифрование каналов передачи данных.

7. Результат
- Рабочая информационная система.
- Интеграция с системой мониторинга транспорта.
- Клиентский портал и аналитические дашборды.
- Полный комплект технической документации.
